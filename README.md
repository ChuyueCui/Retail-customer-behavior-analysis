# Retail customer behavior analysis (Data Analytics Project)

📌 Project Overview  
This project analyzes retail customer behavior using a large-scale transaction dataset (300K+ records).  
The goal is to understand how demographic factors, product preferences, and shipping methods influence purchasing decisions and customer feedback.

🔍 Key Contributions  

- Cleaned and preprocessed large-scale retail data (handling missing values, outliers, duplicates)  
- Conducted hypothesis-driven analysis on customer behavior  
- Applied statistical tests (t-test, chi-square) to validate relationships  
- Built machine learning models (Decision Tree, Random Forest, Regression)  
- Generated business insights to support marketing and product strategy  

📊 Key Insights  

- Age significantly influences purchasing behavior, with middle-aged customers showing higher spending  
- Gender differences exist in product preferences (e.g., electronics vs. clothing)  
- Same-day shipping improves customer satisfaction, especially for time-sensitive products  

🏆 Methods Used  

- Data Cleaning & Feature Engineering  
- Statistical Testing (T-test, Chi-square)  
- Machine Learning (Decision Tree, Random Forest, Regression)  

🛠️ Tools & Technologies  

- Python  
- pandas, NumPy  
- scikit-learn  
- Data Visualization
